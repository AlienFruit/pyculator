# Поддержка Markdown в окне вывода

Для полноценного отображения Markdown в окне вывода результатов необходимо установить две библиотеки:

## Установка библиотек

```bash
pip install markdown tkinterhtml
```

## Что дает полноценный Markdown

После установки библиотек приложение автоматически будет использовать HTML-рендеринг для отображения Markdown, что обеспечивает:

- ✅ Правильные размеры заголовков (H1, H2, H3)
- ✅ Жирный и курсивный текст с правильным форматированием
- ✅ Блоки кода с подсветкой синтаксиса
- ✅ Инлайн код с фоном
- ✅ Списки (маркированные и нумерованные)
- ✅ Таблицы
- ✅ Ссылки
- ✅ Поддержка темной и светлой темы

## Без библиотек

Если библиотеки не установлены, приложение будет использовать упрощенное форматирование через цветовое выделение текста в CTkTextbox.
