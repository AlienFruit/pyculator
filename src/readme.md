# Создание Python калькулятора

## Общая информация
Язык приложения: python.

Это должно быть приложение с графическим интерфейсом. Для интерфейса нужно использовать современную графическую библиотеку для python, которая отображает красивый современнный интерфейс.

В приложении будет основное окно, разбитое на 2 части:
- Окно редактирования, которое будет являться редактором python
- Окно отображения, таб будет отображаться результат выполнения скрипта, в том числе графики

Так же должна быть панель с кнопками сохранения и открытия файла.

Данные должны храниться в json файле.
